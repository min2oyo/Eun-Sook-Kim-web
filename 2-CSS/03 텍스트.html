font

font-family:

  • "Arial Regular", "sans-serif", "돋움", "굴림", "굴림"

font:

  • <font-style><font-variant><font-weight>

    • font-style 속성값을 font 속성값으로 사용
    • font-style이라는 키워드를 사용하는 것이 아님

font-style:

  • normal

  • italic

    • 처음부터 기울어진 글꼴이 디자인 됨
    • 웹에서 주로 사용

  • oblique

    • 원래 글꼴을 기울어지게 표시함

font-variant:

  • normal

  • small-caps

font-weight:

  • 100 ~ 900

  • normal

    • 400
    • 보통 굵기
    • 기본값

  • bold

    • 700
    • 굵게

  • bolder

    • 원래보다 더 굵게

  • lighter

    • 원래보다 더 가늘게

font-size:

키워드

  • xx-small

  • x-small

  • medium

  • larger

  • x-large

  • xx-large

단위

  • 10em

    • 부모 요소에서 지정한 글꼴의 대문자 M의 너비(1em)로 비율값 지정

  • 10rem

    • 문서 시작 부분(root)에서 지정한 크기(1rem)로 비율값 지정

  • 10ex

    • 해당 글꼴의 소문자 x의 높이(1ex)로 비율값 지정

  • 10px

    • 모니터의 1픽셀(1px)로 비율값 지정

  • 10pt

    • point로 불림
    • 일반 문서에서 많이 사용

  • 10%

    • 부모 요소의 글자 크기를 기준으로 계산하여 지정
    • 부모 요소의 글꼴 크기가 단위(10px)로 표현되어 있어야 함
