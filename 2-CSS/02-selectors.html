일반

  • 나중에 작성한 스타일이 적용됨
  • 배경색, 배경 이미지는 상속 안 됨

연결 선택자(조합 선택자, 콤비네이션 선택자, 콤비네이션 셀렉터 등)

하위 요소

  • 특정 요소를 기준으로 그 안에 포함된 요소
  • 자식 요소
    • 현재 요소를 기준으로 바로 한 단계 아래 요소
  • 손자 요소
    • 자식 요소의 한 단계 아래 요소

- descendant selector(하위 선택자)

section p {}
  • 상위요소 하위요소
  • 부모 요소에 포함된 모든 하위 요소에 스타일 적용
  • 자손 선택자라고도 함

- child selector(자식 선택자)

select>p {}
  • 부모요소>자식요소
  • 자식 요소에만 스타일 적용

형제 요소

  • 웹 문서에서 부모 요소가 같음
  • 형 요소
    • 먼저 나오는 요소
  • 동생 요소
    • 나중에 나오는 요소

- adjacent selector(인접 형제 선택자)

h1+p {}
  • 요소1+요소2
  • 요소1과 요소2는 같은 레벨이면서 요소1 이후 가장 먼저 오는 요소2를 선택

- sibling selector(형제 선택자)

h1~p {}
  • 요소1~요소2
  • 모든 형제 요소에 적용

속성 선택자

  • 태그 안에서 사용하는 속성값에 따라 요소를 선택
  • 속성 값의 조건에 따라 특정 부분만 선택하기 쉬워서 상황에 맞는 스타일을 지정하기 쉬움

[속성] 선택자

a[href] {}
  • 해당 속성 존재
  • a 요소 중에 href 속성이 있는 요소

[속성 = 속성값] 선택자

a[target = _blank] {}
  • 해당 속성값 존재
  • a 요소 중에 target 속성값이 _blank인 요소

[속성 ~= 속성값] 선택자

[class ~= button] {}
  • 해당 속성값 포함
  • class값 중에 button이 있는 요소
  • 주의
    • 단어별
    • '-'을 연결한 단어에 스타일을 적용하지 않음
    • 'flat-button', 'buttons' 안 됨

[속성 |= 속성값] 선택자

a[title |= us] {}
  • 해당 속성값 포함
  • title 속성값에 us가 있거나 us-로 연결된 속성값이 있는 a 요소
  • 주의
    • 단어별
    • '-'을 연결한 단어에 스타일을 적용함

[속성 ^= 속성값] 선택자

a[title ^= eng] {}
  • 해당 속성값 시작
  • title 속성값이 eng로 시작하는 a 요소

[속성 $= 속성값] 선택자

[href $= xls] {}
  • 해당 속성값 끝
  • 링크한 파일 이름의 마지막 단어가 xls인 요소

[속성 *= 속성값]

[href *= w3]
  • 해당 속성값 포함
  • href 속성값 중에 'w3'가 포함된 요소
