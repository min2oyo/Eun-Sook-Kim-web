연결 선택자(조합 선택자, 콤비네이션 선택자, 콤비네이션 셀렉터 등)

하위 요소

  • 특정 요소를 기준으로 그 안에 포함된 요소
  • 자식 요소
    • 현재 요소를 기준으로 바로 한 단계 아래 요소
  • 손자 요소
    • 자식 요소의 한 단계 아래 요소

하위 요소 | 하위 선택자(descendant selector)

section p {}
  • 상위요소 하위요소
  • 부모 요소에 포함된 모든 하위 요소에 스타일 적용
  • 자손 선택자라고도 함

하위 요소 | 자식 선택자(child selector)

select>p {}
  • 부모요소>자식요소
  • 자식 요소에만 스타일 적용

형제 요소

  • 웹 문서에서 부모 요소가 같음
  • 형 요소
    • 먼저 나오는 요소
  • 동생 요소
    • 나중에 나오는 요소
